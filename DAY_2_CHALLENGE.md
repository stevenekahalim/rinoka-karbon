## Challenge 1: Change the Accent Color (15 min)

### Steps:
1. Open `css/style.css` in your text editor
2. Go to line 8-22 (the `:root` section)
3. Find this line:
   ```css
   --secondary-color: #00ff88;
   ```
4. Change it to:
   ```css
   --secondary-color: #ff0088;
   ```
5. Save the file
6. Refresh your website in the browser

### What You Should See:
- Green accents become pink/magenta
- Buttons change color
- Hover effects change
- Logo border changes

### Experiment:
Try these colors too:
- Blue: `#0088ff`
- Purple: `#8800ff`
- Orange: `#ff8800`
- Yellow: `#ffff00`

## Challenge 2: Create a New Color Scheme (20 min)

### Your Task:
Make the website blue-themed instead of green-themed.

### Steps:
1. In `css/style.css`, change these variables:
   ```css
   --primary-color: #ffffff;        /* Keep white */
   --secondary-color: #00aaff;      /* Change to blue */
   --accent-color: #00aaff;         /* Change to blue */
   ```

2. Save and refresh

3. Take a screenshot and compare to the green version

### Bonus Challenge:
Create a "fire theme" for charcoal:
- Secondary color: `#ff4500` (orange-red)
- What other colors would match?

## Challenge 3: Change Background Colors (15 min)

### Task:
Make the website lighter (less black).

### Steps:
1. Find these lines in `:root`:
   ```css
   --dark-color: #0a0a0a;
   --darker-color: #000000;
   --card-bg: #1a1a1a;
   ```

2. Change to:
   ```css
   --dark-color: #2a2a2a;      /* Lighter */
   --darker-color: #1a1a1a;     /* Lighter */
   --card-bg: #3a3a3a;          /* Lighter */
   ```

3. Save and refresh

### Question:
- Does it still look "modern" or does it need to be very dark?

## Understanding What Happened

### You just learned:
1. **CSS Variables** - Define once, use everywhere
2. **Color Codes** - Hex codes like #00ff88
3. **Cascade Effect** - One change affects many elements
4. **Design Consistency** - Using variables keeps design unified
